Nosferatu Pure CSS Animation

Written by Aomi Bora in Blog

A Cool Animation made using Pure CSS. Made by Josetxu.

The codes are separated in two parts, so copy all two parts combine them and after that run the code. You can use two separate files for HTML, CSS or you can just use one index.html file.
